    • 构造样式规则

    样式表中的每条规则都有两个主要部分：选择器(selector) 和 声明块 (declaration block)

    选择器(selector)： 决定哪些元素收到影响

    声明块 (declaration block)：由一个或者多个属性－值对组成。即，每个属性－值对构成一条声明。

    inheritance 继承

    当某个元素应用CSS属性时，这些属性不仅会影响该元素，还会影响其下的分支元素

    以下是哪些属性会被继承：

  • 文本
  • color (颜色，a元素除外)
  • direction
  • font
  • font-family
  • font-size
  • font-style 用于设置斜体
  • font-variant 用于设置小型大写字母
  • font-weight
  • letter-spacing
  • line-height
  • text-align 用于设置对齐方式
  • text-indent
  • text-transform 用于修改大小写
  • visibility
  • white-space
  • word-spacing

  • 列表
  • list-style 列表样式
  • list-style-image 用于为列表指定定制的标记
  • list-style-position 用于确定列表标记的位置
  • list-style-type 用于设置列表的标记

  • 表格
  • border-collapse 用于控制表格相邻单元格的边框是否合并为单一边框
  • border-spacing 用于指定表格边框之间的空隙大小
  • caption-side 用于设置表格标题的位置
  • empty-cells 用于设置是否显示表格中的空单元格

  • 其他
  • cursor 鼠标指针
  • quotes 用于指定引号样式
